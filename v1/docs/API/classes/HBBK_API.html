\HBBK_API

Class HBBK_API; Wrapper for hbbk-ilias.de as an API.

This Class Enables the User of this API to do certain things with hbbk-ilias.de in one call, and doesn't need multiple steps to get to the target file (e.g. a timetable).

authenticate()

authenticate(String  $password) : boolean

HBBK_API authenticator.

Makes a call to hbbk-ilias.de POSTing the client's password, which is the parameter, and the username, which was a parameter of the constructor, to authenticate the user in order to use the API. Returns TRUE if the authentication was successful and FALSE if it wasn't, in that case it should be retried.

Parameters

String $password

The Users Password for Authenticating with hbbk-ilias.de.

getTimetable()

getTimetable(String  $week, String  $class) : mixed|void

HBBK_API getTimtable.

Makes a call to hbbk-ilias.de taking week and class as parameter to get the current timetable of that week for that class. Right now it only returns the HTML it got from the call, in future versions the API will be able to put out HTML/JSON with a parameter for the format of the return.

Parameters

String $week

The desired week offset the timetable should be of.
String $class

The desired class the timetable should be of.

Returns

mixed|void

logout()

logout() : boolean

HBBK_API logout.

Makes a call to hbbk-ilias.de and logs the user out, so the system can accept him accessing the timetable through the app, or through the traditional way. This function then also deletes the cookie of the user because they are not needed anymore. Return whether the logout was successful or not.

getClassNum()

getClassNum(  $class) : false|string

HBBK_API getClassNum.

This function checks the class given by the user against either one of the arrays and tries to find the matching UUID (001..140) If the class couldn't be found it returns FALSE, otherwise it'll return the classes UUID as a String.

Parameters
